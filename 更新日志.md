## 2026-07-14 — 两层数据架构落地(规范层持久化 + 按 URDF 派生本体层)

**动机**:数据复用。原管线在采集时就把人手 21 点 retarget 成 inspire 12 关节并丢掉原始点(有损不可逆),只存了本体层 `lerobot_ds`;换机器人厂家/本体无法复用,等于「只留编译产物、丢了源码」。

**改动**:
- 新增 `sim/probe_canonical_feats.py`:先验证 LeRobotDataset 能存/读回扁平 float32 特征(`hand_keypoints(63,)`、`wrist_pose(7,)`)+ video。**通过**(数值精确往返),故规范层用扁平向量存(多维 (21,3)/(4,4) 本版本未验证,规避)。
- 新增 `sim/build_canonical.py`:视频 → `sim/out/canonical_ds`。每帧存 ego RGB(video)+ `observation.hand_keypoints`(21×3 MANO 米→扁平 63)+ `observation.wrist_pose`(7=平移3+四元数4,相机系)+ task。**不 retarget、不平滑**。复用 `SingleHandDetector`+`estimate_wrist_pose`。产出 710 帧、0 丢检。

## 2026-07-14 — 手腕朝向稳定化(方案3轻量版:出平面降权 + 残差门限)

**背景**:rerun 可视化(见 07-13)暴露出「人手视频近乎静止、仿真臂却大幅晃」。

**诊断(实验坐实)**:
- 臂**只由手腕朝向驱动**(位置锚定 home 固定,`build_robot_traj.py` 只有相对朝向在变)。
- 手腕朝向相对首帧漂 **0~43.4°(均值 21°)**;帧间抖动 SavGol 后仅 0.86°/帧 → **主因是低频漂移,不是高频抖**。
- 分解:漂移 **91% 落在出平面**(绕相机 X/Y,手掌法向倾斜,单目深度估不准),面内(绕光轴 Z,图像内滚转)只有 0~6.6° → 基本是真手势。
- 旁证:手腕 z 位置在「静止」下漂 26cm = 单目深度噪声铁证。
- 增益实验:缩放相对朝向重跑 IK,臂摆幅合计 scale 1.0=184° / 0.5=68° / 0.25=32° → **臂≈1:1 跟朝向**,治源头是唯一杠杆。
- 否掉的路:IK 零空间正则(方案2)只收肩部 joint1/2,动不了正干朝向活的 joint5/7(k_null=0.1 时 joint7 反涨、IK 收敛 710→470);回环修正不适用(管线无开环积分,朝向每帧独立绝对估计,无累积状态)。

**改动**:
- 新增 `sim/wrist_stabilize.py`:
  - `gate_outliers(quats, gate_deg)`：帧间旋转增量超阈值则沿测地线限幅,剔单帧跳变。
  - `attenuate_out_of_plane(Rs, alpha, ref)`：相对参考帧,把出平面(绕相机 X/Y)朝向分量乘 alpha,面内(绕光轴 Z)全保留。相机系约定:光轴 = +Z。
- `sim/build_robot_traj.py`:接入上面两步(符号对齐 → 门限 → SavGol → 出平面衰减 → IK);加 CLI `--oop-alpha`(默认 0.4)、`--gate-deg`(默认 8)。稳定化默认开;`--oop-alpha 1.0 --gate-deg 0` 复现基线。
- `sim/nero_kin.py`:`ik()` 加 `q_rest`/`k_null` 零空间正则参数(默认 0 向后兼容)。经实验非主因,保留为可选旋钮。
- `sim/replay_rerun.py`:关节曲线改为按轨迹标签分组(`joints/<label>/...`),A/B 时同图叠看 raw vs stab。

**验证**:臂运动总幅度 **184.6° → 57.4°(降到 31%)**;joint5 63.5→17.0、joint7 49.8→11.5;**IK 仍 710/710 全收敛**(不像零空间正则掉到 470);门限剔 9 帧;面内真手势保留。

**产物**:`sim/out/robot_traj.pkl`(稳定化默认版,已替换)、`sim/out/robot_traj_raw.pkl`(基线,供 A/B)、`sim/out/replay_ab.rrd`(rerun A/B)。

**未做/后续**:完整 RTS 平滑器 / 因子图 + 各向异性可观测性协方差(用前缩调制 alpha),待 Femto 深度到货(每像素置信通道 + 出平面歧义本身变小)再建。残余出平面偏置需深度或先验封顶,单目衰减不能归零。
